About Website

Intended User's

Employers. Also, family and friends so that you have some idea of what I do professionally.

Logging In

You must use your google login/passwd. No worries, this website is just using goggle's free login facility. Even if you don't login, you can still browse items and add them to your shopping cart. It is only when you "check out" that you will need to login.

Using website

Navigate and click on what ever you want. If this website in not easily navigated, then I failed at making it simple because that's how good websites should function. Also, no worries when you "place order", the website does nothing other than saying that your order was a success. Nothing is charged to you because I did not build that into this website. Just think of it as pretending to buy something.

Technical's of website

This website was written in Angular 6 typeScript. It uses a Firebase realtime NoSQL database; although I'm used to relational databases/SQL which have a RESTful/JSON (CRUD) API.
This website also uses Bootstrap 4 components and Font Awesome icons, as well as some other software that you can see in the package.json depencies from the sourse code for this website that is stored in GitHub at URL: https://github.com/jerry-carney/jerry-shop
